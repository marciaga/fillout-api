# Fillout assignment

### Technologies used

- Express
- Typescript
- Jest
- Yup
- Eslint
- Prettier

### Development

Install Node modules:

```sh
$ npm install
```

Run the dev server (file changes restart the server automatically using `nodemon`):

```sh
$ npm run dev
```

Run the tests:

```sh
$ npm run test
```

Build for production:

```sh
$ npm run build
```
